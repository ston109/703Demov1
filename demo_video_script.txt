703DEMOV1 Demo Video Script
Target length: under 3 minutes at about 120 words per minute
Estimated spoken length: about 350 words

0:00-0:20 Hook and Problem
Screen: Show the shopping website home/product list.

Script:
Online stores lose revenue when shoppers hesitate after adding products to cart. This system predicts risk, identifies the reason, and safely recovers the sale.

0:20-0:45 Product Overview
Screen: Show MockMart, login/register, product page, cart.

Script:
This is not a chatbot. The user never types a prompt. The system observes product views, cart actions, checkout exits, shipping pages, reviews, similar products, coupon attempts, and partial checkout.

0:45-1:30 Live Demo Flow
Screen: Log in, add product to cart, open cart, leave cart, browse shipping/reviews/similar products.

Script:
I log in, add a product, and open the cart. Entering the cart does not reduce the score. Risk scoring starts when I leave the cart with products still inside. Then I check shipping, reviews, and cheaper products. These signals suggest shipping concern, uncertainty, price sensitivity, or comparison hesitation.

Screen: Switch to AGI Monitor.

Script:
In the AGI Monitor, we see the session, score, events, actions, feedback, evaluation, and error logs. The score is scoped to the account and current cart. If the cart is cleared, analysis pauses. If checkout is completed, the cycle ends or resets for remaining products.

1:30-2:10 Agent Architecture
Screen: Show AGI Monitor and briefly show backend structure if useful.

Script:
The backend is a compound AI system. Events pass through the shopping backend, which injects the cart snapshot. The AGI backend updates a world model, builds belief state, retrieves memory, checks goals, plans actions, applies safety, and executes a website action.

The LLM is not the brain. Gemini is only a bounded helper for low-score risk calibration. State, memory, planning, safety, tools, feedback, and evaluation are maintained by the system itself.

2:10-2:45 Business Value, Safety, and Cost
Screen: Show support highlight, recommendation strip, banner, or small demo incentive.

Script:
The business value is conversion recovery. The agent can clarify shipping, highlight support, recommend products, or offer a demo incentive capped at five percent. It cannot auto-purchase, modify prices, collect payment data, or send emails or SMS. Visible actions use risk thresholds, cooldown, and duplicate filtering.

Token cost is controlled because Gemini is called only in a narrow scoring path. Output is capped at 10 tokens and must return fixed JSON. If Gemini is missing, slow, or invalid, deterministic fallback runs immediately. If Gemini returns later, the backend can apply cart-scoped correction without letting the LLM control actions.

2:45-3:00 Closing
Screen: Show evaluation panel.

Script:
This is a real autonomous AI system: it receives input, models the environment, diagnoses the problem, chooses a safe action, executes it in the website, collects feedback, and evaluates performance. The result is a commercially useful AGI-style agent for reducing cart abandonment.
